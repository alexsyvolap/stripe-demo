## Install

- Copy `.env.example` to `.env`
- Insert to `.env` Stripe secret and public key
- Insert to `.env` connect to database
- Insert to `.env` APP_NAME and APP_URL
- Run `composer install`
- Run `php artisan key:generate`
- Run `php artisan migrate`
- Run `php artisan db:seed` (echo new user data in console)
- Run `php artisan serve`
